# 第一次安装虚拟机电脑被迫重启

```
	
	运行 VMware 时，不时的会遇到在重启虚拟机时导致宿主机重启  
	
	只要 VMware 的 CPU 设置中，“每个处理器的内核数量”为“1”就不会导致该问题，而大于“1”就有可能导致重启，并且近期多个版本的 VMware 都存在这个问题，可能是CPU的 bug，也可能是 VMware 的 Bug，
	临时的解决办法就是将“每个处理器的内核数量”设置为“1”，将不会触发虚拟机重启导致宿主机重启的现象（或者关闭主板上 BIOS 内 CPU 的超线程设置应该也可以解决）
	
	
	主要解决方法删掉所有虚拟机 清除VmWare 完全卸载它 去官网下载最新版 修复正常
	
```
